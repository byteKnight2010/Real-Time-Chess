# Real-Time Chess

Chess, but both players move at the same time.

No waiting for your opponent. No turns. Just pressure.

---

## Download

Grab the latest release from the [Releases](../../releases) page.

Two options are available:
- **Installer** — recommended, sets up a Start Menu shortcut and uninstaller
- **Portable ZIP** — extract anywhere and run

Windows x64 only.

---

## Audio

Background music [currently unimplemented] and move sound effects are included. Volume for each can be adjusted independently, or mute everything at once from the menu.

If no audio device is available at launch, the game still runs; audio fails silently, and a note is written to the log.

---

## After a Game

Once a game ends, you can open the analysis screen to review what happened before heading back to the menu or jumping straight into a rematch.

---

## Running from the ZIP

Extract the ZIP somewhere, make sure the `assets` folder sits next to `RealTimeChess.exe`, and run it. No install needed.

A log file (`chess_debug.log`) is written next to the executable on each launch — check it first if something goes wrong.

---
